== How It Works

.1. Define GraphQL schema.
[source,graphql]
----
type Query {
  greet(name: String!): String!
}
----

.2. Write Java application code.
[source,java]
----
@Component
public class QueryResolver {
  public String greet(String name) {
    return "Hello " + name;
  }
}
----

.3. Build and run.
[source,sh]
----
mvn spring-boot:run
----
